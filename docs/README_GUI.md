# DDOJY 高/中熵合金工作流

更完整的参数说明和换电脑环境配置见 [使用教程.md](使用教程.md)。软件里也内置了“教程与帮助”页，可以直接浏览同样的详细说明。

这个程序是一套工作流式的高/中熵合金设计器：启动时先选择工作目录和 LAMMPS 环境或可执行文件，再进入建模模块或 LAMMPS 接口模块。程序会优先识别 conda 里的 LAMMPS wheel 运行时，也支持直接输入 `python.exe`、环境目录或 `lmp.exe`。建模页可以生成结构并导出 LAMMPS `.data`，LAMMPS 页可以生成输入脚本、按核数并行运行并绘制热力学结果，还支持更多场景预设、更多力场选择、pair_style / pair_coeff 覆盖和自定义模板编辑。现在 LAMMPS 页还带有一个可直接查看、继续编辑并保存的 in 文件预览区，方便先生成再手工改脚本，然后一键运行。当前版本在导出最终 alloy 模型前会自动清理极近距离原子对，LAMMPS 运行时会把脚本和日志放到 `process/`，把弛豫结果写到 `relaxation/relaxed.data`、`relaxation/relaxed.restart` 和 `relaxation/trajectory.lammpstrj`，把生产态结果写到 `md/production.data`、`md/production.restart` 和 `md/trajectory.lammpstrj`。动态场景前会自动做一轮结构最小化，必要时先松弛盒子，再进入正式 NVT/NPT/NPH 运行；NVE 场景还会先做短程限位预热，减少初始失稳。如果 log 里出现丢原子、危险邻居表更新或类似错误，界面也会给出诊断提示。工作目录、LAMMPS 路径、核数和最近使用的场景会保存到 `config.json`。教程与帮助页也已经内置到软件里，方便直接浏览完整操作说明。

## 使用方法

1. 运行 [run_gui.bat](../run_gui.bat)，或者直接执行 `python hea_mea_designer.py`。
2. 启动后先确认工作目录和 LAMMPS 环境或可执行文件；所有生成脚本、日志和中间文件默认写到该目录。
3. 在“工作台”页先看当前环境，再进入建模模块或 LAMMPS 接口模块。
4. 在“配方”页直接输入 5 元或 7 元 HEA 配方，也可以用预设按钮加载常见组合。
5. 在“掺杂”页添加置换、空位、表面吸附或间隙插入，常用的 H、O、Si、Cu 模板已经预置；控制参数会按不同操作解释为表面距离或界面带宽。
6. 在“梯度晶粒”页调整模型尺寸、晶粒尺寸、层数、随机扰动和排布方式；这里的晶粒模型基于泰森多边形（Voronoi）生成，默认采用六角错位 seed 排布，左侧还提供了摘要复制、节点 CSV 导出和研究报告导出按钮。
7. 在“单晶/多晶建模”页可以在三维多晶、纳米粉末和单晶之间切换，左侧输入参数、右侧查看单一预览窗口；三维多晶支持规则网格和随机播种，纳米粉末支持球形、立方、椭球、圆柱和八面体等形状，单晶支持晶面、晶界和位错模板。该页顶部集成了 Atomsk 高级构型后处理，可对当前模型执行格式标准化、复制超胞、镜像孪晶和镜像合并双层，并保存 `.atomsk.txt` 命令报告；如果输出路径与当前源模型冲突，可以点击“自动命名输出”，执行后程序也会自动准备下一次安全输出路径。每个建模面板都带有 `.data` 导出按钮。
8. 在“LAMMPS接口”页选择场景、数据文件、力场和势函数文件后可以生成 `in.lammps`，也可以直接启动 conda 运行时或本地 `lmp.exe`。页面上方可以选核数，程序会优先用 `mpiexec` 做多核运行；如果运行时没有 GPU package，GPU 选项会自动禁用。输出组织区会显示 `process/`、`relaxation/` 和 `md/` 的默认文件位置，并支持一键接续弛豫数据或生产态数据。运行结果会在界面右侧自动绘图；如果你要做更复杂的输入，也可以直接在高级模板框里改脚本。
9. 在“梯度输出”页的模型载入区里，可以切换到二维六边形多晶、二维随机多晶、孪晶、多相和 K-S 取向模型；这里主要负责当前梯度多晶源文件的输出和归档。
10. 在“裂纹”页可以选择中心裂纹或边缘裂纹，也可以在矩形和椭圆/圆之间切换；长度和开口两项在椭圆模式下分别对应长轴和短轴。
11. 点击“一键生成全部”可以把几何、裂纹、掺杂和合金配方串起来一次完成。

如果你把程序打包成 exe，程序会把资源按 `data/`、`models/`、`docs/` 归类，旧的 exe 输出会在重新打包时被清理掉。

## 打包成 exe

先安装 PyInstaller：

```bash
python -m pip install pyinstaller
```

然后运行 [package.bat](../package.bat)。生成的可执行文件会在 `dist\HEA_MEA_Designer` 目录里，同时会把默认要用的 [data/final.lmp](../data/final.lmp)、[data/build.bat](../data/build.bat)、[data/gradient.txt](../data/gradient.txt)、[data/aluminium.xsf](../data/aluminium.xsf) 以及新集成的模型文件夹和说明文档一起复制进去。

如果你选的是 `.cfg` 或 `.xsf` 模型，程序会先借助 Atomsk 转成临时 LAMMPS，再进入配方、建模和裂纹流程。

## 说明

当前程序最终处理的仍是 `Atoms # atomic`，也就是 `id type x y z` 这种格式；如果选的是 CFG 或 XSF，程序会先借助 Atomsk 转成临时 LAMMPS，再继续后面的流程。如果以后你的 LAMMPS 文件格式变了，需要再扩展解析器。
