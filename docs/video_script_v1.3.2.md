# HEA/MEA Designer 分享视频脚本（v1.3.2）

## 视频定位

- 目标时长：3 到 5 分钟。
- 目标观众：材料模拟、分子动力学、合金建模、LAMMPS 初学者到进阶用户。
- 核心信息：这是一个面向高/中熵合金建模与 LAMMPS 前处理的桌面工具，重点是降低重复建模成本、保留复现实验记录，并把 Atomsk 后处理、掺杂、裂纹、梯度晶粒和 LAMMPS 输入组织到同一工作流里。
- 表述边界：不要宣称“自动得到真实材料结论”。应明确它是建模和模拟准备工具，最终科学结论仍需要势函数选择、参数收敛、实验或更高精度方法交叉验证。

## 片头（0:00-0:20）

画面：打开 `HEA_MEA_Designer.exe`，停留在首页和左侧工作流导航。

旁白：
“大家好，这个小工具叫 HEA/MEA Designer，目前演示版本是 v1.3.2。它主要解决一个实际问题：在做高熵、中熵合金分子动力学模拟时，建模、掺杂、缺陷、LAMMPS 输入文件和结果目录往往分散在不同脚本里，复现和批量修改都比较费时间。这个工具把这些步骤整合成一个桌面工作流。”

屏幕字幕：
“HEA/MEA Designer v1.3.2：高/中熵合金建模与 LAMMPS 前处理”

## 工作流总览（0:20-0:55）

画面：依次指向左侧导航：配方、掺杂、梯度晶粒、单晶/多晶建模、裂纹、LAMMPS 接口。

旁白：
“左侧是完整流程。可以先定义合金配方，比如 FeCoNiCrMn；再做置换、空位、间隙和表面吸附类掺杂；随后可以生成梯度晶粒、二维或三维多晶、纳米粉末、单晶和缺陷模型。后面还有裂纹模板和 LAMMPS 接口，用于生成输入脚本、组织弛豫和生产模拟目录。”

屏幕动作：
1. 切换到“配方”页，展示元素比例输入。
2. 切换到“掺杂”页，展示掺杂表格。
3. 切换到“单晶/多晶建模”页，停在 Atomsk 后处理区域。

## Atomsk 建模增强（0:55-1:45）

画面：在“单晶/多晶建模”页顶部展示“Atomsk 高级构型后处理”。

旁白：
“v1.3.2 重点优化了 Atomsk 后处理入口。这里支持格式标准化和 wrap、复制超胞、镜像孪晶，以及镜像后沿指定轴合并成双层构型。工具会直接显示将要执行的 Atomsk 命令，便于检查和复现。”

屏幕动作：
1. 操作选择“复制超胞”。
2. 设置复制倍数，例如 X=2、Y=1、Z=1。
3. 点击“预览命令”。
4. 展示命令预览和输出路径。

旁白：
“这里有一个很重要的细节：输出文件不能覆盖源文件。以前连续执行时，如果当前源模型刚好也是上一次输出，容易出现路径冲突。v1.3.2 已经修复这个问题：可以点击‘自动命名输出’，执行完成后程序也会自动准备下一次安全输出路径，并写出 `.atomsk.txt` 命令报告。”

屏幕字幕：
“关键修复：自动避让源文件，避免 Atomsk 输出覆盖当前模型”

## 建模示例（1:45-2:35）

屏幕动作：
1. 点击“执行 Atomsk 后处理”。
2. 展示状态栏完成提示。
3. 打开输出目录或展示生成的 `.atomsk.txt` 报告。

旁白：
“这类报告不是装饰信息，而是为了科研复现。以后写论文、做批量对比，至少能回头确认模型是怎么从源文件变换出来的。”

旁白：

屏幕字幕：
“科研边界：建模工具不替代势函数验证、收敛测试和实验/高精度交叉验证”
