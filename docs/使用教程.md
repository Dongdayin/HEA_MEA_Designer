# HEA/MEA Designer 高/中熵合金设计器使用教程

这份教程对应当前目录里的 HEA/MEA Designer 高/中熵合金设计器。它的核心思路是：先选择工作目录和 LAMMPS 环境或可执行文件，再生成梯度多晶几何、单晶/多晶/纳米粉末结构，随后可选地导出 LAMMPS `.data` 或进入 LAMMPS 接口页做输入生成和异步运行。程序现在还内置了一个“教程与帮助”页，你可以直接在软件里浏览这份教程和快速说明。当前版本会优先识别 conda 里的 LAMMPS wheel 运行时，也支持直接选择 `python.exe`、环境目录或 `lmp.exe`。

## 1. 你会得到什么

这个程序不是传统意义上的“单一材料建模器”，而是一个流程工具，能把以下几步串起来：

1. 生成梯度多晶节点和多晶几何。
2. 对现有 LAMMPS 结构做元素替换，生成高熵合金或中熵合金版本。
3. 生成中心预裂纹或边缘预裂纹。
4. 输出可直接用于后续模拟的 LAMMPS 文件。

界面分成工作台、配方、掺杂、晶粒、建模、裂纹、输出、LAMMPS 接口和教程与帮助页。

## 2. 目录里哪些文件最重要

- [hea_mea_designer.py](../hea_mea_designer.py)：主程序源码。
- [hea_mea_gui.py](../hea_mea_gui.py)：兼容入口，保留给旧启动方式使用。
- [run_gui.bat](../run_gui.bat)：双击启动界面。
- [package.bat](../package.bat)：把程序打包成 exe。
- [data/final.lmp](../data/final.lmp)：默认的结构输入文件。
- [models/](../models)：界面“科研模型库”实际引用的精选预设模型。
- [docs/README_GUI.md](README_GUI.md)、[docs/verification.md](verification.md)：当前版本的快速说明和验证说明。

打包后的目录会按 `data/`、`models/`、`docs/` 分组，只包含当前版本运行和教学需要的文件。`generated/` 是程序运行后自动生成的工作目录，不属于仓库或 release 包内容。

## 3. 快速上手

最短流程如下：

1. 双击 [run_gui.bat](../run_gui.bat)。
2. 第一次启动时先选工作目录和 LAMMPS 环境或可执行文件；这些设置会写入 `config.json`，下次打开时会自动恢复。你可以直接选 conda 环境目录、`python.exe` 或 `lmp.exe`，程序会自动解析成实际可运行的 LAMMPS 路径。
3. 在“配方”页填入元素和权重，例如 5 元高熵合金可以用 `Fe20Co20Ni20Cr20Mn20`。
4. 在“晶粒”页调整模型尺寸、晶粒尺寸、层数、随机扰动。
5. 在“裂纹”页选择是否要中心裂纹或边缘裂纹。
6. 在“输出”页选择输入文件、输出文件和 Atomsk 路径。
7. 点击“一键生成全部”。

如果你只是想给现有结构换元素，不改几何，也可以只在“配方”页填好，然后点“仅生成合金”。

## 4. 配方页参数解释

“配方”页负责决定最终是什么元素体系，也就是高熵合金或中熵合金的组成。

### 4.1 直接输入

可以直接输入字符串，例如：

```text
Fe20Co20Ni20Cr20Mn20
```

也可以输入：

```text
Co33.3Cr33.3Ni33.4
```

程序会自动识别元素符号和权重，并做归一化。也就是说，数值不一定必须刚好加到 100，程序会自动按比例分配。

### 4.2 元素表每一列是什么意思

| 列名 | 含义 | 怎么调 |
|---|---|---|
| 色块 | 元素颜色预览 | 只是视觉提示，不影响计算 |
| 元素 | 元素符号 | 例如 Fe、Co、Ni、Cr、Mn |
| 权重/摩尔分数 | 该元素在配方中的比例 | 数值越大，最终该元素原子数越多 |
| 质量 | 原子质量 | 通常自动填充，也可手动覆盖 |
| 清除 | 清空这一行 | 方便重新输入 |

### 4.3 预设按钮怎么用

- `3 元预设`：Co-Cr-Ni，适合中熵合金演示。
- `5 元预设`：Fe-Co-Ni-Cr-Mn，适合常见 HEA。
- `7 元预设`：Al-Co-Cr-Fe-Mn-Ni-Ti，适合更复杂的多组元方案。

点预设后，你可以再手工改权重。比如你想把 Fe 提高一点，就把 Fe 那一行的权重调大，其他元素相应调小。

### 4.4 如何调配更合理

- 想做标准 5 元等原子高熵合金，直接用每个元素 20。
- 想做 7 元等原子体系，直接用每个元素 14.2857 左右。
- 想做某个元素偏多的非等原子体系，就把它的权重提高即可。
- 如果输入了重复元素，程序会自动合并成同一个元素的总权重。

### 4.5 什么时候用这一页

这一页只决定“最终原子类型分配”，不改变几何坐标。

如果你已经有一个做好的多晶结构，只想把它变成 HEA/MEA，就只用这一页也能完成。

## 5. 掺杂页参数解释

“掺杂”页负责给现有结构添加置换、空位、表面吸附和间隙插入等操作。它适合做氢、氧、硅等元素的掺杂，也适合做带空位缺陷的模型。

### 5.1 这个页怎么用

1. 如果你想让掺杂真正生效，先勾选“启用掺杂”。
2. 选择一个模板，然后点“套用模板”，程序会自动填好常见参数。
3. 如果你想自己改，也可以直接改表格里的每一行。
4. 最后在“输出”页点“仅生成合金”或者“一键生成全部”，掺杂会一起写进最终文件。

### 5.2 常用模板

- `H 间隙掺杂`：在体内插入少量氢原子。
- `O 表面吸附`：在上表面吸附氧原子。
- `空位缺陷`：删除一部分原子形成空位。
- `Si 置换掺杂`：把一部分原子换成硅。
- `界面带掺杂`：在中间区域做带状掺杂。

### 5.3 每一列是什么意思

| 列名 | 含义 | 怎么调 |
|---|---|---|
| 元素 | 要掺杂进去的元素符号 | 例如 H、O、Si、Cu |
| 操作 | 置换、空位、吸附、间隙 | 决定是替换、删除还是新增原子 |
| 区域 | 整体、上表面、下表面、界面带 | 决定掺杂发生在结构哪里 |
| 数量 | 掺杂比例或数量 | 根据“单位”解释为百分比或个数 |
| 单位 | 比例% 或 个数 | 比例% 会按当前可用原子自动换算 |
| 控制参数 | 距离或带宽 | 吸附时表示表面距离，界面带时表示带宽 |

### 5.4 什么时候用这一页

- 你要做氢扩散、氧吸附、表面污染或空位缺陷。
- 你要给某个区域单独掺杂，不想影响整个配方。
- 你要在最终输出里同时保留多晶、裂纹和掺杂信息。

## 6. 晶粒页参数解释

“晶粒”页负责生成梯度多晶几何。你可以把它理解成“控制晶粒大小和模型尺寸”的地方。

### 6.1 模型尺寸

| 参数 | 含义 | 怎么调 |
|---|---|---|
| 宽度 (Å) | 模型在 X 方向的总宽度 | 增大后，整个模型更宽 |
| 高度 (Å) | 模型在 Y 方向的总高度 | 增大后，整个模型更高 |

通常情况下，宽度和高度决定了你最终几何框有多大。

如果你点“从源文件同步”，程序会读取当前源文件的盒子尺寸，把宽高填成源文件的尺寸。

### 6.2 晶粒控制

| 参数 | 含义 | 怎么调 |
|---|---|---|
| 第一层晶粒数 | 最底层的晶粒个数 | 越大，底层晶粒越细 |
| 目标晶粒尺寸 (Å) | 直接按目标尺寸反推第一层晶粒数 | 填了以后会优先于第一层晶粒数 |
| 层间变化 | 每一层增加或减少多少个晶粒 | 越大，晶粒梯度越强 |
| 层数 | 一共有多少层梯度层 | 越大，梯度越长 |
| 种晶晶格常数 (Å) | 生成种晶时的晶格参数；切换到 BCC/HCP 时会自动填入推荐值 | 对铝默认 4.046 合理 |
| 随机扰动 | 控制节点偏离规则位置的程度 | 越小越规整，越大越不规则 |
| 排布方式 | 六角错位或分层直列 | 默认选六角错位，更接近二维多晶论文中的 Voronoi 多边形 |
| 周期结构 | 是否做上下对称/周期布局 | 想要周期性可打开 |
| 随机种子 | 固定随机性 | 固定后结果可复现 |

### 6.3 这几个参数怎么配合

#### 想让晶粒更细

- 增大“第一层晶粒数”。
- 或者减小“目标晶粒尺寸”。
- 宽度不变时，第一层晶粒数越大，每个晶粒就越小。

#### 想让晶粒更粗

- 减小“第一层晶粒数”。
- 或者增大“目标晶粒尺寸”。

#### 想让梯度更明显

- 增大“层间变化”。
- 增加“层数”。

#### 想让结构更规整

- 把“随机扰动”调小，常见范围是 0.00 到 0.05。
- 固定“随机种子”。

#### 想让结构更随机

- 把“随机扰动”调大，但不要太大，否则节点会偏移过头，几何会乱。

### 6.4 推荐调参思路

如果你是第一次做，建议先用这样的起点：

- 宽度：500 Å
- 高度：1200 Å
- 第一层晶粒数：2 或 4
- 层间变化：1
- 层数：5
- 种晶晶格常数：4.046 Å
- 如果选的是 HCP，软件会另外给出 c/a，默认 1.633；HCP 生成时会同时带上 a 和 c。
- 随机扰动：0.01 到 0.03

如果你要更细的晶粒，就把第一层晶粒数调大，例如 6、8、10。

### 6.5 周期结构是什么意思

打开周期结构后，程序会把节点按对称方式扩展，得到上下更对称的梯度布局。

适合下面几种情况：

- 你想让模型上下更接近周期边界。
- 你想做重复层状结构。
- 你想让边界看起来更规整。

如果你只是想要一个简单的梯度多晶，不追求对称性，可以关闭它。

### 6.6 这一页实际生成什么

点“一键生成全部”时，这一页会先生成：

- `generated/seed.xsf`
- `generated/gradient.txt`
- `generated/final.lmp`

这些文件是几何中间结果，后面裂纹和配方都是在这个基础上继续处理。

### 6.7 科研工具

晶粒页左侧现在提供了三类直接可用的科研工具：

1. 复制摘要：把当前晶粒参数、节点范围和晶粒尺寸统计复制到剪贴板，便于写论文或做组会记录。
2. 导出节点 CSV：把当前几何节点导出成表格文件，方便后续做脚本分析、画图或和别的参数集对比。
3. 导出研究报告：把摘要、层级数据和节点数据打包成一份文本报告，适合保留实验记录。

这些功能的目标是让晶粒参数扫描、结果整理和论文复现更直接，不需要再手工抄参数。

### 6.8 三维多晶建模

“建模”页里的“三维多晶”不再暴露梯度参数，只需要输入长、宽、高，并选择规则网格或随机播种模式，程序会自动推导三维种子布局并生成一个三维 Voronoi 多晶源文件。生成结果会直接继承当前配方和掺杂设置，适合做体材料、多尺度对比和后续裂纹流程。

### 6.9 高熵合金纳米粉末建模

“建模”页里的“高熵合金纳米粉末”可以直接控制粉末大小、个数和形状，支持球形、立方、椭球、圆柱和八面体等形状预设。生成后会直接继承当前配方和掺杂设置，并得到可用于后续裂纹流程的 LAMMPS 源文件。

### 6.10 单晶与缺陷建模

“建模”页里的“单晶”用于单晶、晶面、晶界和位错示意模型。你可以选择晶面取向、缺陷类型、缺陷角度和位错核半径；默认勾选“继承上一步结果”时，会把当前配方和掺杂继续应用到单晶结构上，方便后续直接接入裂纹或输出流程。

### 6.11 LAMMPS 接口

“LAMMPS接口”页用于把当前结构接到模拟环节。你可以先选场景，再指定数据文件、力场、势函数和脚本输出路径，点击生成即可写出 `in.lammps`；如果本机已经配置好 LAMMPS 运行时，还能直接在界面里异步运行并在右侧查看温度、总能量等热力学曲线。页面顶部可以设置环境/程序路径、核数和 GPU 选项：如果当前运行时带有 MPI，程序会优先用 `mpiexec` 按核数并行；如果没有 GPU package，GPU 选项会自动禁用。现在这个页面还提供了更多场景预设、更多力场选项、pair_style / pair_coeff 覆盖项，以及一个可直接编辑的自定义模板框；生成后的 in 文件也会显示在脚本预览区，可以继续修改后保存并运行。由于页面内容较多，界面本身也支持滚动查看。若你在建模页已经生成了新结构，程序会自动把 LAMMPS 数据文件指向最新结果。

## 7. 裂纹页参数解释

这一页负责做“几何预裂纹”。它的做法很直接：在指定位置删除一块原子区域，形成初始裂纹或缺口。

### 7.1 裂纹模式

| 模式 | 含义 | 适用场景 |
|---|---|---|
| 无 | 不加裂纹 | 正常完整模型 |
| 中心裂纹 | 在模型中心挖出裂纹 | 研究断裂起裂、裂纹扩展 |
| 边缘裂纹 | 从边界往内挖出裂纹 | 研究边裂、缺口敏感性 |

### 7.2 裂纹方向

| 方向 | 含义 |
|---|---|
| 水平 | 裂纹沿 X 方向展开 |
| 垂直 | 裂纹沿 Y 方向展开 |

### 7.3 边缘方向

这个选项只在“边缘裂纹”模式下有意义。

- 水平裂纹时可以选左或右。
- 垂直裂纹时可以选上或下。

### 7.4 裂纹长度和开口宽度

| 参数 | 含义 | 怎么调 |
|---|---|---|
| 裂纹长度 (Å) | 裂纹沿主方向伸展的长度 | 越大，裂纹越长 |
| 开口宽度 (Å) | 裂纹空隙的厚度 | 越大，裂纹缝越宽 |

这两个参数都是以 Å 为单位。

如果你在“裂纹形状”里选的是“椭圆/圆”，那么这两个参数就分别表示椭圆的长轴和短轴；两个值设成一样时就是圆。矩形模式下，它们仍然表示裂口的长边和短边。

一般建议：

- 裂纹长度不要大到接近整个模型尺寸，否则会把太多原子删掉。
- 开口宽度要结合晶粒尺寸来调，太小看不出来，太大容易破坏模型完整性。

### 7.5 这是不是 Atomsk 的 crack 物理场裂纹

不是。

当前程序实现的是几何预裂纹，也就是“删掉一块原子”，适合做初始缺口、预裂纹或简化的断裂模型。

如果你以后想做带应力强度因子的连续介质 crack 场，再接 Atomsk 的 crack 选项会更合适。

### 7.6 这一页什么时候用

- 你想做断裂力学相关模拟。
- 你想给模型加一个起裂缺口。
- 你想比较中心裂纹和边缘裂纹的影响。

## 8. 输出页参数解释

“输出”页负责把流程真正串起来。

### 8.1 Atomsk 路径

如果你要生成几何，程序需要找到 `atomsk.exe`。

你可以：

1. 把 Atomsk 加到系统 PATH。
2. 或者在这里直接浏览选择 `atomsk.exe`。

如果你只是对现有 LAMMPS 结构做配方替换，Atomsk 不一定非要参与；但只要你要生成几何、转换 `.cfg/.xsf` 或执行 Atomsk 后处理，就必须有 Atomsk。

### 8.2 源文件

默认是 [data/final.lmp](../data/final.lmp)。

你也可以换成其他 LAMMPS data 文件，但要满足一个前提：

- Atoms 段必须是 `id type x y z` 这种 `atomic` 风格。

如果文件格式不同，程序就读不进去。

### 8.3 输出文件

默认输出到 `generated/final_HEA.lmp`。

你可以改成你想要的名字，比如：

- `generated/final_MEA.lmp`
- `generated/test_7component.lmp`

### 8.4 4 个动作按钮

| 按钮 | 做什么 |
|---|---|
| 仅生成几何 | 只做梯度多晶，不做配方替换 |
| 仅应用裂纹 | 在当前源文件上挖裂纹 |
| 仅生成合金 | 只做元素重新分配 |
| 一键生成全部 | 按“几何 -> 裂纹 -> 合金”顺序一次完成 |

## 9. 推荐工作流程

### 情况 A：你想做一个新的 HEA 梯度多晶

1. 在“配方”页输入 5 元或 7 元配方。
2. 在“晶粒”页调宽度、高度、第一层晶粒数、层间变化、层数。
3. 在“裂纹”页选择是否要预裂纹。
4. 在“输出”页检查 Atomsk 路径、输出文件。
5. 点击“一键生成全部”。

### 情况 B：你已有几何，只想换成另一套元素

1. 载入或保留当前结构文件。
2. 在“配方”页填新配方。
3. 点击“仅生成合金”。

### 情况 C：你只想先做缺口测试

1. 先生成几何。
2. 在“裂纹”页调裂纹位置和大小。
3. 再继续做合金赋值或后处理。

### 科研模型库

软件已经把你提供的模型文件夹集成进来了。界面里打开“输出”页，就能在“科研模型库”里直接切换。

| 模型 | 依据 | 适合研究 |
|---|---|---|
| 默认梯度泰森多边形多晶 | Atomsk 的 polycrystal/Voronoi 晶粒 | 梯度晶粒、晶界统计、裂纹起裂 |
| 二维六边形多晶 | 规则六边形晶粒对照 | 理想晶界、规则网格对比 |
| 二维随机多晶 | 随机泰森多边形/Voronoi | 随机晶粒分布、统计分析 |
| 二维梯度孪晶多晶 | 梯度层状孪晶 | 孪晶层厚、层间梯度 |
| 倾斜孪晶多晶 | 倾斜孪晶界面 | 倾角效应、界面位错 |
| 预存孪晶多晶 | 预置孪晶 | 固定孪晶边界对比 |
| 双相多晶 | Cu/W 双相结构 | 相界、失配、双相界面 |
| K-S取向多晶 | K-S 取向关系 | 取向匹配、相界演化 |

如果你选的是 `.cfg` 或 `.xsf` 模型，程序会先用 Atomsk 转成临时 LAMMPS，再继续做配方和裂纹处理。

## 10. 换电脑后怎么配置环境

这部分最关键。根据你是“只运行”还是“继续开发/打包”，分三种情况。

### 方案 A：只把程序拿到另一台电脑上运行

如果你已经打包好了 exe，最省事的做法是：

1. 把整个 `dist\HEA_MEA_Designer` 文件夹拷到新电脑。
2. 保留文件夹里的这些文件：
   - `HEA_MEA_Designer.exe`
   - `_internal/`
   - `data/final.lmp`
   - 当前 release 内置的精选模型预设和说明文档
3. 双击 exe 运行。

如果你只做“仅生成合金”，通常就够了。

如果你还要生成几何或裂纹相关流程，新电脑上仍然需要安装 Atomsk。

### 方案 B：要在新电脑上运行源码版

如果你要直接运行源码，而不是 exe，建议这样配置：

1. 安装 Python 3.11 x64。
2. 把整个项目文件夹复制过去，最好连同 `.venv` 一起复制。
3. 如果 `.venv` 已经在，`run_gui.bat` 会优先使用它。
4. 如果没有 `.venv`，新建一个：

```powershell
python -m venv .venv
```

5. 激活环境：

```powershell
.\.venv\Scripts\Activate.ps1
```

6. 如果你只运行程序，通常不需要额外第三方库，因为代码本身只用标准库。
7. 如果你要重新打包，再安装 PyInstaller：

```powershell
python -m pip install pyinstaller
```

8. 双击 [run_gui.bat](../run_gui.bat) 即可。

### 方案 C：要在新电脑上做几何生成和裂纹生成

这种情况除了 Python 之外，还要安装 Atomsk。

建议做法：

1. 安装 Atomsk。
2. 把 `atomsk.exe` 加到 PATH，或者记住它的完整路径。
3. 打开程序后，在“输出”页里直接浏览选择 `atomsk.exe`。

只要程序能找到 Atomsk，就可以做：

- 生成种晶
- 生成梯度多晶
- 做预裂纹

### 方案 D：要在新电脑上重新打包

如果你要从源码重新打包，推荐这样：

```powershell
Set-Location "E:\DESK TOP\二维梯度多晶"
.\.venv\Scripts\Activate.ps1
python -m pip install pyinstaller
```

然后运行 [package.bat](../package.bat)。

程序会在 `dist\HEA_MEA_Designer` 里生成新的 exe，并把默认结构、精选模型预设、README、教程和验证说明一起拷贝进去。历史教程、视频脚本、Atomsk 中间输出和本机运行输出不会进入分发包。

## 11. 常见问题

### 问题 1：程序提示找不到 Atomsk

处理办法：

- 在“输出”页浏览选择 `atomsk.exe`。
- 或者把 Atomsk 加入系统 PATH。

### 问题 2：读不进 LAMMPS 文件

通常是因为输入文件不是 `Atoms # atomic` 格式。

当前版本只支持 `id type x y z`。

### 问题 3：裂纹把原子删光了

说明裂纹长度或开口太大了。把这两个参数调小，再试一次。

### 问题 4：晶粒太细或太粗

调这个顺序最直接：

- 想细一点，就增大第一层晶粒数，或者减小目标晶粒尺寸。
- 想粗一点，就减小第一层晶粒数，或者增大目标晶粒尺寸。

### 问题 5：结果每次都不一样

这是因为随机种子不同。把“随机种子”固定住，就能复现同一结果。

## 12. 一个实用建议

如果你刚开始调参，不要同时改太多变量。

推荐顺序是：

1. 先固定配方。
2. 再调宽高。
3. 再调第一层晶粒数或目标晶粒尺寸。
4. 再调层间变化和层数。
5. 最后再加裂纹。

这样更容易看出每个参数的作用。
